# GroupBox 📚👥

GroupBox is a modern full-stack collaborative learning platform designed to help students and instructors work together efficiently on academic group projects. The platform provides structured tools for communication, contribution tracking, peer evaluation, and team management, making collaboration more transparent, organized, and productive.

---

## 🚀 Features

### 👥 Group Collaboration

- Create and manage academic project groups
- Organized collaborative workspace for teams
- Smooth and intuitive navigation experience
- Real-time group activity updates

### 📊 Contribution Tracking

- Monitor individual member participation
- Track completed tasks and submissions
- Improve accountability within teams

### 🏆 Leaderboards

- View top-performing groups and students
- Encourage healthy academic competition
- Performance-based ranking system

### 📝 Peer Evaluation

- Evaluate group members fairly
- Transparent feedback and assessment process
- Improve teamwork and collaboration quality

### 🔔 Notifications

- Receive updates on assignments and activities
- Stay informed about announcements and deadlines

### 🔐 Authentication & Authorization

- Secure user authentication
- Role-based access for:
  - Learners
  - Instructors
  - Administrators (if applicable)

### 📱 Responsive Design

- Fully responsive modern UI
- Optimized for desktop, tablet, and mobile devices

## 🛠 Tech Stack

### Frontend

- Context API / Custom Hooks

### Backend

- Node.js

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/GroupBox.git
cd GroupBox
```

---

### 2. Install dependencies

#### Frontend

```bash
cd frontend
npm install
```

#### Backend

```bash
cd backend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

## ▶️ Run the Project

### Start Backend Server

```bash
cd backend
npm run dev
```

### Start Frontend

```bash
cd frontend
npm run dev
```
